# 방법
- 용어가 보이면 전부 용어 사전에 기록
- 각 용어에 대한 설명은 파일로 기록

- 필요하면 스크린샷 이미지 추가
- 내용이 길어지면 나눈다
- 내용이 길어지지 않도록 노력한다
