# CIDR 블록 : CIDR(Classless Inter-Domain Routing)

---
- CIDR은 [기본 IP 주소] / [네트워크 마스크 숫자] 형태로 표기합니다.
- 기본 IP 주소: 네트워크가 시작되는 지점입니다. (예: 10.0.0.0)
- VPC에서 허용되는 마스크 크기는 
  - /16(최대) 에서 
  - /28(최소) 사이입니다.

---
- 10.0.0.0/16: 65,536개의 IP 주소 제공 (대규모 네트워크용)
- 10.0.0.0/24: 256개의 IP 주소 제공
- 10.0.0.0/28: 16개의 IP 주소 제공 (가장 작은 단위)
