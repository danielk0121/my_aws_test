# 네트워크 마스크(Prefix)

---
- IP 주소는 총 32비트로 구성
- 이 32비트 중 "왼쪽부터 몇 번째 자리까지를 네트워크 주소(고정 값)로 쓸 것인가"를 결정

---
- 10.0.0.0/8 (가장 넓은 범위)
  - 고정 비트: 8비트 (첫 번째 덩어리 10이 고정됨)
  - IP 범위: 10.0.0.0 ~ 10.255.255.255
- 172.16.0.0/12 (중간 범위)
  - 고정 비트: 12비트 (첫 번째 172와 두 번째 덩어리의 일부가 고정됨)
  - 의미: 두 번째 숫자가 16부터 31까지만 같은 네트워크로 묶입니다.
  - IP 범위: 172.16.0.0 ~ 172.31.255.255

---
- 2진수로 풀어서 생각하기
  - IP 주소 172.16.0.0의 두 번째 덩어리인 16을 8비트 2진수로 바꾸면 다음과 같습니다.
  - 16 = 0 0 0 1 | 0 0 0 0
  - 여기서 /12 라는 것은 앞의 12비트를 고정한다는 뜻입니다.
    - 두 번째 덩어리: 0 0 0 1 (앞의 4비트만 추가로 고정)
    - 고정된 부분: 172. + 0 0 0 1 (이 부분은 절대 안 변함)
    - 변할 수 있는 부분 (뒤의 4비트): 0 0 0 0 ~ 1 1 1 1
    - 그러므로 16 + [0, 15] = [16, 31]
  - 172.0.0.0/12: 고정 비트가 172. + 0 0 0 0 이므로 172.0 ~ 172.15 범위가 됩니다.
  - 172.16.0.0/12: 고정 비트가 172. + 0 0 0 1 이므로 172.16 ~ 172.31 범위가 됩니다.
